Project Description

This project is a predator vs prey ecosystem simulation built using Python, Pygame, and Pygwidgets. 
The simulation shows animals moving around in a virtual environment. Prey move randomly and try to survive, while predators chase prey to stay alive. 
The simulation will allow the user to change settings such as speed, population size, and reproduction rate to see how the ecosystem changes over time.
The goal of the project is to create a visual simulation that demonstrates how different variables affect the balance between predators and prey.

Features

Moving animals on the screen using Pygame
Predators chase prey
Prey move randomly
Animals can reproduce over time
Predators lose energy if they do not eat
User controls using Pygwidgets sliders and buttons
Reset / start / pause simulation
Adjustable settings:
   Number of predators
   Number of prey
   Speed
   Reproduction rate
   Hunger rate
